# Command: Whitelist

:::tip Description
`Whitelists a user from Antinuke`
:::

## Usage

```
whitelist <add|remove|show> <user>
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: whitelist
- Alias 2: wl
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- Administrator
+++ Bot Permissions
- ManageGuild
- ViewChannel
- SendMessages
- EmbedLinks
+++
