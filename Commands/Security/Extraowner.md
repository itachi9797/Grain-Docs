# Command: Extraowner

:::tip Description
`Extraowners for your server`
:::

## Usage

```
extraowner <add|remove|show> <user>
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: extraowner
- Alias 2: eo
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

+++ Bot Permissions

- ViewChannel
- SendMessages
