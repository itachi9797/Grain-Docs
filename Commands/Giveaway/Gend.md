# Command: Gend

:::tip Description
`Ends a giveaway.`
:::

## Usage

```
gend <giveaway message id>
```

## Category

_**Giveaways**_

:::info Aliases
- Alias 1: giveawayend
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ManageGuild
- ViewChannel
- SendMessages
+++ Bot Permissions
- ViewChannel
- SendMessages
+++
