# Command: Gdelete

:::tip Description
`Deletes a giveaway.`
:::

## Usage

```
gdelete <giveaway message id>
```

## Category

_**Giveaways**_

:::info Aliases
- Alias 1: giveawaydelete
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ManageGuild
- ViewChannel
- SendMessages
+++ Bot Permissions
- ViewChannel
- SendMessages
+++
