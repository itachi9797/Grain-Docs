# Command: Greroll

:::tip Description
`Rerolls a giveaway.`
:::

## Usage

```
greroll <giveaway message id>
```

## Category

_**Giveaways**_

:::info Aliases
- Alias 1: giveawayreroll
- Alias 2: reroll
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ManageGuild
- ViewChannel
- SendMessages
+++ Bot Permissions
- ViewChannel
- SendMessages
+++
