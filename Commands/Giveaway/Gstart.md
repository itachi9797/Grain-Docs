# Command: Gstart

:::tip Description
`Start a giveaway.`
:::

## Usage

```
.gstart <time> <winner count> <prize>
```

## Category

_**Giveaways**_

:::info Aliases
- Alias 1: giveawaystart
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::

## Permissions

+++ User Permissions
- ManageGuild
- ViewChannel
- SendMessages
+++ Bot Permissions
- ViewChannel
- SendMessages
+++
