# Command: Delete

:::tip Description
`Delete a user.`
:::

## Usage

```
delete <user>
```

## Category

_**Fun**_

:::info Aliases
- Alias 1: Delete
- Alias 2: Trash
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ViewChannel
- SendMessages
+++ Bot Permissions
- ViewChannel
- SendMessages
