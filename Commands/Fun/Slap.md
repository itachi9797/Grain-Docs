# Command: Slap

:::tip Description
`Slap a user.`
:::

## Usage

```
slap <user>
```

## Category

_**Fun**_

:::info Aliases
- Alias 1: slap
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::
