# Command: RoleIcon

:::tip Description
`Change a role icon`
:::

## Usage

```
roleicon <role> <icon>
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: roleicon
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- Manage Roles
+++ Bot Permissions
- ViewChannel
- SendMessages
- EmbedLinks
- Manage Roles
+++
