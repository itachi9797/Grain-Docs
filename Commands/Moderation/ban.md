# Command: Ban

:::tip Description
`Bans a User from server`
:::

## Usage

```
ban <user> [reason]
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: fuckoff
- Alias 2: fuckban
:::

:::caution Cooldown
This command has a cooldown of 20 seconds.
:::

## Permissions

+++ User Permissions
- BanMembers
+++ Bot Permissions
- BanMembers
- ViewChannel
- SendMessages
- EmbedLinks
+++
