# Command: Unhide

:::tip Description
`Hide A Channel`
:::

## Usage

```
unhide <channel>
unhide all
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: unvanish
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ManageChannels
+++ Bot Permissions
- ManageChannels
- ViewChannel
- SendMessages
- EmbedLinks
+++
