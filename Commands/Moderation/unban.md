# Command: Unban

:::tip Description
`UnBan A User`
:::

## Usage

```
Unban <user> [reason]
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: uban
:::

:::caution Cooldown
This command has a cooldown of 20 seconds.
:::

## Permissions

+++ User Permissions
- BanMembers
+++ Bot Permissions
- BanMembers
- ViewChannel
- SendMessages
- EmbedLinks
+++
