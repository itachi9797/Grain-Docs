# Command: Lock

:::tip Description
`Locks a perticular channel`
:::

## Usage

```
lock <channel>
lock all
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: lock
- Alias 2: lockdown
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ManageChannels
+++ Bot Permissions
- ManageChannels
- ViewChannel
- SendMessages
- EmbedLinks
+++
