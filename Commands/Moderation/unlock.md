# Command: Unlock

:::tip Description
`Unlocks A Channel`
:::

## Usage

```
Unlock <channel>
Unlock all
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: unlock
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ManageChannels
+++ Bot Permissions
- ManageChannels
- ViewChannel
- SendMessages
- EmbedLinks
+++
