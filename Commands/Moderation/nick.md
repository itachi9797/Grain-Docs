# Command: Nickname

:::tip Description
`ChangeA User Nickname`
:::

## Usage

```
Nickname <user>
```

## Category

_**Moderation**_

:::info Aliases
- Alias 1: Nickname
- Alias 2: Nick
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- Manage Nicknames
+++ Bot Permissions
- ViewChannel
- SendMessages
- EmbedLinks
- Manage Nicknames
+++
