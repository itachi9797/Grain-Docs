# Command: Welcome

:::tip Description
`Will Welcome the user to your server`
:::

## Usage

```
.welcome <add/remove> <channel>
```

## Category

_**Welcome**_

:::info Aliases
- Alias 1: welcome
:::

:::caution Cooldown
This command has a cooldown of 3 seconds.
:::

## Permissions

+++ User Permissions
- ManageGuild
+++ Bot Permissions
- ViewChannel
- SendMessages
- EmbedLinks
+++
