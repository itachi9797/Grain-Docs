# Command: Autorole

:::tip Description
`Gives role to user when he joins the server`
:::

## Usage

```
autorole <bot|human> <add|remove> <role>
```

## Category

_**Welcome**_

:::info Aliases
- Alias 1: autoroles
:::

:::caution Cooldown
This command has a cooldown of 5 seconds.
:::

## Permissions

+++ User Permissions
- ManageGuild
+++ Bot Permissions
- ViewChannel
- SendMessages
- EmbedLinks
+++
